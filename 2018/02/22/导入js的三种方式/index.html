导入js的三种方式
2018-02-22 19:11:43 # 前端

1 行内导入方式（不推荐，不安全）

2 内嵌式

3 外链式

如果把 script 元素放在 head 元素里，待浏览器经过下载、解析、执行完 js 文件之后，才会呈现页面内容。如果一个页面需要很多个 js 文件，那么浏览器可能会出现短暂的“白板”，用户体验不好。所以，我们应该把 js 文件放在 </body> 的前面，这样浏览器会先展现页面给用户，type 属性可以不写。

4 注意点

内嵌导入和外链导入不能合并到一起。
